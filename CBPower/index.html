CBPower

About

CBPower is a website that uses the ELO rating system to rank college basketball teams based on their wins and losses. The ELO system was created to be a more rigorous system for ranking chess players and has been adopted by several other groups such as League of Legends and DOTA to rank teams and players.

The website was built to scrape scores for games, put them into a database, perform the ELO rating system on the teams, then rank them based on ELO ratings. This data was then presented to users on the screen in the form on a list of teams and their rankings in descending order.

We then added functionality to allow users to see graphs of a teams progression throughout the year using highcharts.

Technologies Used

  • Python
      Python was used to perform the web-scraping and ELO ratings
  • Flask/Jinja
      Flask was used as the framework to generate the pages of the site. Jinja was used to make html templates.
  • Bootstrap
      Bootstrap was used for the general layout and format of the site.
  • Highcharts
      Highcharts is an open-source Javascript library we used to create graphs of team progression.
